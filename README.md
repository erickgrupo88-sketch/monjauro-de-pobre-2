# Monjauro de Pobre 2

Clone exato da pagina XQuiz, servido por um proxy Node.js para manter o mesmo HTML, design, etapas do quiz, botoes, imagens e provas sociais.

## Link online

```text
https://monjaurodepobre.vercel.app/
```

No Windows, abra:

```text
ABRIR-SITE-ONLINE.cmd
```

## Rodar localmente

No Windows, abra:

```text
RESETAR-E-ABRIR.cmd
```

Ou pelo terminal:

```bash
npm start
```

Depois acesse:

```text
http://localhost:8090/?utm_source=instagram&utm_campaign=bio
```

## Publicar online

Use uma hospedagem que rode Node.js, como Render, Railway, Replit ou VPS. O GitHub Pages puro nao executa `server.js`.
